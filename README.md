# NetoLabs

The public website for [NetoLabs](https://netolabs.dev), a venture builder founded by Luiz Neto.

NetoLabs conceives ideas, assembles founding teams, develops products, and operates new companies until they can stand on their own.

## Stack

- Next.js 16 with App Router
- React 19
- Tailwind CSS 4
- Motion
- Phosphor Icons

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run build
```
